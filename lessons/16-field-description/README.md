# Accessible Cross-Browser Forms

## `16-field-description`

Sometimes it's beneficial to add more context for a form field than can be provided by the label. In this lesson we'll create the `.field-description` class to enable an additional field description. And we'll learn how to ensure users of assistive technology also receive the description in context. This text span can also be used to provide validation messages.
